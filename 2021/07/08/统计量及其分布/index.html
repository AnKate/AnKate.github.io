统计量及其分布

Posted on 2021-07-08

统计量及其分布

数理统计绪论

样本与总体

总体：研究对象的全体，可分为有限总体和无限总体。

样本：从总体中随机抽取的个体。

关系：对样本进行抽象可以得到总体。

从总体中抽取样本的方法很多，最常用的简单随机抽样具有两个要求：

  • 代表性：总体中每一个个体都有同等机会被选入样本，即每个样本\(x_i\)需要与总体\(X\)服从相同的分布。
  • 独立性：每一个样本的取值不会影响其他样本的取值，即每个样本相互独立。

以常见的测量问题为例，对一个物理量\(\mu\)进行重复测量，记测量结果为\(X\)，误差为\(\varepsilon\).

测量结果可以看做物理量与测量误差的叠加，即： \[ X=\mu+\varepsilon \] 称\(\mu\)为参数，\(\varepsilon\)是随机变量，关于总体分布的假定主要是关于\(\varepsilon\)服从的分布的假定，有以下几种情况：

  • 由于中心极限定理，假定随机误差\(\varepsilon\sim N(0,\sigma^2)\)，则总体分布\(X\sim N(\mu, \sigma^2)\)，此时未知参数为\(\mu,\sigma\)
  • 知道\(\varepsilon\)服从的分布的方差时，假定其服从\(\varepsilon \sim N(0,\sigma_0^2)\)，则总体分布\(X \sim N(\mu, \sigma_0^2)\)，此时未知参数仅有\(\mu\)
  • 无法认定误差服从正态分布时，假定\(\varepsilon\)的分布关于0对称，则总体分布\(X\)称为非参数分布

经验分布函数

定义：设\(x_1,x_2,...,x_n\)是取自总体分布函数\(F(x)\)的样本，若将样本观测值由小到大进行排列，记为\(x_{(1)},x_{(2)},...,x_{(n)}\)，称排列后的样本为有序样本，用有序样本定义如下函数： \[ F_n(x)= \begin{cases} 0&,~x<x_{(1)}\\\\ k/n&,~x_{(k)}\leq x < x_{(k+1)},~k=1,2,...,n-1\\\\ 1&,~x\geq x_{(n)} \end{cases} \] 称\(F_n(x)\)为该样本的经验分布函数。

经验分布函数是一个分布函数：

  • 由定义过程可知，\(F_n(x)\)满足右连续性与单调性（非减）
  • \(F(-\infty)=0,~F(\infty)=1\)，满足有界性

经验分布函数也可以使用示性函数写成如下形式： \[ F_n(x)=\frac{1}{n}\sum^n_{i=1}I\{x_i\leq x\} \]

格列文科定理：

设\(x_1,x_2,...,x_n\)是取自总体分布函数\(F(x)\)的样本，\(F_n(x)\)是经验分布函数，当\(n\rightarrow \infty\)时，有 \[ P(\sup_{-\infty<x<\infty}|F_n(x)-F(x)|\rightarrow 0)=1 \] 即当样本量足够大时，经验分布函数会逐渐接近总体分布函数。

经验分布函数的期望与方差：

记随机变量\(Y \sim F_n(x)\)，则：

\(E(Y)=\displaystyle\frac{1}{n}\sum^n_{i=1}x_i=\overline x\)（样本均值）

\(Var(Y)=\displaystyle\frac1n\sum^n_{i=1}(x_i-\overline x)^2=s_n^2\)（样本方差）

统计量

样本均值与样本方差

定义：设\(x_1,x_2,...,x_n\)是取自某总体的样本，其算术平均值称为样本均值，用\(\overline x\)表示，即 \[ \overline x = \frac1n \sum^n_{i=1}x_i \] 关于样本均值\(\overline x\)的平均偏差平方和称为样本方差，用\(s_n^2\)表示，即 \[ s_n^2=\frac1n \sum^n_{i=1}(x_i-\overline x)^2 \] 定理：

若总体的均值为\(\mu\)，方差为\(\sigma^2\)，\(x_1,x_2,...,x_n\)为样本：

\(E(\overline X)=\mu, ~Var(\overline X)=\displaystyle\frac{\sigma^2}n\)

\(E(s^2_n)=\displaystyle\frac{n-1}{n}\sigma^2\)

令\(s^2=\displaystyle\frac{1}{n-1}\sum^n_{i=1}(x_i-\overline x)^2\)，称\(s^2\)为无偏方差，其期望\(E(s^2)=\sigma^2\).

通常使用无偏方差作为样本方差。

样本均值的分布：

设\(x_1,x_2,...,x_n\)是来自某个总体的样本，\(\overline x\)为样本均值。

  • 若总体分布为\(N(\mu,\sigma^2)\)，则\(\overline x \sim N(\mu, \displaystyle \frac{\sigma^2}{n})\)
  • 若总体分布未知，但已知期望\(\mu\)和方差\(\sigma^2\)，则在样本量较大时，\(\overline x\)的近似分布为正态分布，记为\(\overline x \sim AN(\mu, \displaystyle \frac{\sigma^2}{n})\)

样本矩

定义：设\(x_1,x_2,...,x_n\)是样本，\(k\)为正整数，则统计量 \[ a_k=\frac{1}{n}\sum^n_{i=1}x_i^k \] 称为样本\(k\)阶原点矩；

统计量 \[ b_k=\frac{1}{n}\sum^n_{i=1}(x_i-\overline x)^k \] 称为样本\(k\)阶中心矩。

相关统计量：

  • 样本偏度：反映了样本数据与对称性的偏离程度和偏离方向 \[ \hat{\beta_s}=\frac{b_3}{b_2^{\frac32}} \]

  • 样本峰度：反映了总体分布密度曲线在其峰值附近的陡峭程度和尾部粗细的统计量 \[ \hat{\beta_k}=\frac{b_4}{b_2^2}-3 \]

次序统计量

定义：设\(x_1,x_2,...,x_n\)是取自总体\(X\)的样本，\(x_{(i)}\)称为该样本的第\(i\)个次序统计量，它的取值是将样本观测值由小到大排列后得到的第\(i\)个观测值。其中\(x_{(i)}=\min\{x_1,x_2,...,x_n\}\)称为该样本的最小次序统计量，\(x_{(n)}=\max\{x_1,x_2,...,x_n\}\)称为该样本的最大次序统计量。

单个次序统计量的分布：

设总体\(X\)的密度函数为\(p(x)\)，分布函数为\(F(x)\)，\(x_1,x_2,...,x_n\)为样本，则第\(k\)个次序统计量\(x_{(k)}\)的密度函数为 \[ p_k(x)=\frac{n!}{(k-1)!(n-k)!}(F(x))^{k-1}(1-F(x))^{n-k}p(x) \] 特别地，有 \[ p_1(x)=n\cdot(1-F(x))^{n-1}p(x) \]

\[ p_n(x)=n\cdot(F(x))^{n-1}p(x) \]

结论：总体分布为\(U(0,1)\)时，第\(k\)个次序统计量的密度函数为 \[ p_k(x)=\frac{n!}{(k-1)!(n-k)!}x^{k-1}(1-x)^{n-k},~0<x<1 \] 即\(Be(k,n-k+1)\)，从而有\(E(x_{(k)})=\displaystyle\frac{k}{n+1}\).

多个次序统计量的分布：

设总体\(X\)的密度函数为\(p(x)\)，分布函数为\(F(x)\)，\(x_1,x_2,...,x_n\)为样本，则第\(i,j\)个次序统计量\(x_{(i)},x_{(j)}\)的联合密度函数为 \[ p_{ij}(x,y)=\frac{n!}{(i-1)!(j-i-1)!(n-j)!}(F(x))^{i-1}(F(y)-F(x))^{j-i-1}(1-F(y))^{n-j}p(x)p(y) \]

样本极差：\(R=x_{(n)}-x_{(1)}\)

样本分位数

样本中位数：记为\(m_{0.5}\)，定义 \[ m_{0.5}= \begin{cases} x_{(\frac{n+1}2)}&,n为奇数\\ \\ \frac12(x_{(\frac{n}{2})}+x_{(\frac{n+1}{2})})&,n为偶数 \end{cases} \] 样本分位数：样本的\(p\)分位数\(m_p\)如下定义 \[ m_p= \begin{cases} x_{(\lfloor np+1 \rfloor)}&,np不是整数\\ \\ \frac12 (x_{(np)}+x_{(np+1)})&,np是整数 \end{cases} \] 定理：设总体密度函数为\(p(x)\)，\(x_p\)为其\(p\)分位数，\(p(x)\)在\(x_p\)处连续且\(p(x_p)>0\)，则当\(n\rightarrow\infty\)时，样本\(p\)分位数\(m_p\)的渐近分布为 \[ m_p\sim N(x_p,~\frac{p(1-p)}{n\cdot p^2(x_p)}) \] 特别地，对于样本中位数，有 \[ m_{0.5}\sim N(x_{0.5},\frac{1}{4n\cdot p^2(x_{0.5})}) \]

三大抽样分布

卡方分布

定义：设\(X_1,X_2,...,X_n\)独立同分布于正态标准分布\(N(0,1)\)，则\(\chi^2=X_1^2+X_2^2+...+X_n^2\)的分布称为自由度为\(n\)的\(\chi^2\)分布，记为\(\chi^2\sim \chi^2(n)\).

卡方分布是特殊的伽马分布：\(\chi^2(n)=Ga(\frac n2,\frac12)\)

密度函数： \[ p(y)=\frac{(\frac{1}{2})^{\frac{n}{2}}}{\Gamma(\frac{n}{2})}y^{\frac{n}{2}-1}e^{-\frac{y}{2}} \] 根据伽马分布的期望与方差，当\(Y\sim \chi^2(n)\)时，有\(E(Y)=2n,Var(Y)=2n\).

定理：设\(x_1,x_2,...,x_n\)是来自正态总体\(N(\mu,\sigma^2)\)的样本，其样本均值为 \[ \overline x = \frac1n \sum^n_{i=1}x_i \] 样本方差为 \[ s^2=\frac{1}{n-1}\sum^n_{i=1}(x_i-\overline x)^2 \] 则有：

  • \(\overline x\)和\(s^2\)相互独立
  • \(\overline x \sim N(\mu, \displaystyle\frac{\sigma^2}{n})\)
  • \(s^2\)的分布：\(\displaystyle\frac{(n-1)s^2}{\sigma^2}\sim \chi^2(n-1)\)

F分布

定义：设随机变量\(X_1 \sim \chi^2(m),~X_2 \sim \chi^2(n),~X_1\)与\(X_2\)独立，则称 \[ F=\frac{X_1/m}{X_2/n} \] 的分布是自由度\(m\)与\(n\)的\(F\)分布，记为\(F\sim F(m,n)\)，其中\(m\)称为分子自由度，\(n\)称为分母自由度。

两个总体的方差比较：

记\(x_1,x_2,...,x_m\)是取自总体\(X\sim N(\mu_1, \sigma_1^2)\)，记\(y_1,y_2,...,y_n\)是取自总体\(Y\sim N(\mu_2, \sigma_2^2)\)，两样本相互独立抽取，有样本方差： \[ s_x^2=\frac{1}{m-1}\sum^m_{i=1}(x_i-\overline x)^2 \]

\[ s_y^2=\frac{1}{n-1}\sum^n_{i=1}(y_i-\overline y)^2 \]

根据前述定理，可知： \[ \frac{(m-1)s_x^2}{\sigma_1^2}\sim \chi^2(m-1) \]

\[ \frac{(n-1)s^2_y}{\sigma_2^2}\sim\chi^2(n-1) \]

则根据\(F\)分布的定义，可以构造 \[ F=\frac{s_x^2/\sigma_1^2}{s_y^2/\sigma_2^2}\sim F(m-1,n-1) \]

t分布

定义：设随机变量\(X_1\)与\(X_2\)相互独立，且\(X_1\sim N(0,1)\)，\(X_2\sim \chi^2(n)\)，则称 \[ t=\frac{X_1}{\sqrt{X_2/n}} \] 是自由度为\(n\)的\(t\)分布，记为\(t\sim t(n)\)

两个总体的均值比较：

记\(x_1,x_2,...,x_m\)是取自总体\(X\sim N(\mu_1, \sigma^2)\)，记\(y_1,y_2,...,y_n\)是取自总体\(Y\sim N(\mu_2, \sigma^2)\)，两样本相互独立抽取，有样本均值： \[ \overline x \sim N(\mu_1, \frac{\sigma^2}m) \]

\[ \overline y \sim N(\mu_2, \frac{\sigma^2}{n}) \]

两样本均值相互独立，则有 \[ \overline x - \overline y \sim N(\mu_1-\mu_2, (\frac{1}{m}+\frac{1}{n})\sigma^2) \] 进行标准化得： \[ \frac{(\overline x - \overline y ) - (\mu_1-\mu_2)}{\sqrt{(\frac{1}{m}+\frac{1}{n})\sigma^2}}\sim N(0,1) \] 引入概念合方差\(s_w^2\)，有 \[ s_w^2=\frac{\displaystyle \sum^m_{i=1} (x_i-\overline x)^2 + \sum^n_{j=1}(y_j-\overline j)^2}{m+n-2} \] 根据\(t\)分布定义，可以构造得 \[ \frac{(\overline x - \overline y ) - (\mu_1-\mu_2)}{s_w^2\sqrt{\frac1m+\frac1n}}\sim t(m+n-2) \]

充分统计量

统计量的作用是对样本进行简化，但实际上，统计量包含的信息可能远小于样本的分布（如经验分布函数），譬如统计量（矩、次序统计量）相同时，两个分布可能不同。

样本的分布中会含有有用的信息（参数），而充分统计量中会包含有关参数的信息，与样本分布能够传递的信息相同。

定义：设\(x_1,x_2,...,x_n\)是来自某个总体的样本，总体分布函数为\(F(x;\theta)\)，统计量\(T=T(x_1,x_2,...,x_n)\)；如果在给定\(T\)的取值后，\(x_1,x_2,...,x_n\)的分布与\(\theta\)无关，即\(F(x_1,x_2,...,x_n|T)\)与\(\theta\)无关，则称\(T\)为\(\theta\)的充分统计量。

该定义在已知充分统计量，进行该统计量的充分性证明时需要用到。

常见分布的充分统计量：设\(x_1,x_2,...,x_n\)是来自如下分布的样本，则有参数的充分统计量：

  • 泊松分布：\(T=\overline x\)是\(\lambda\)的充分统计量
  • 方差已知的正态分布\(N(\mu,\sigma_0^2)\)：\(T=\overline x\)是\(\mu\)的充分统计量
  • 方差未知的正态分布\(N(\mu,\sigma^2)\)：\(T=(\overline x, s^2)\)是\(\mu,\sigma^2\)的充分统计量
  • 均匀分布\(U(a,b)\)：\(T=(x_{(1)},x_{(n)})\)是\(a,b\)的充分统计量

充分统计量未知时，则无法从定义出发进行判定，此时可以借助因子分解定理进行充分统计量的寻找。

因子分解定理：设总体概率函数\(f(x;\theta)\)，\(x_1,x_2,...,x_n\)为样本，则\(T=T(x_1,x_2,...,x_n)\)（维数不限）是充分统计量的充要条件是：存在两个函数\(g(t,\theta)\)和\(h(x_1,x_2,...,x_n)\)使得对任意的\(\theta\)和任一组观测值\(x_1,x_2,...,x_n\)，有 \[ f(x_1,x_2,...,x_n)=g(T(x_1,x_2,...,x_n),\theta)h(x_1,x_2,...,x_n) \]

指数型分布族：（正态、二项、负二项、泊松、指数、伽马、贝塔） \[ p_{\theta}(x)=c(\theta)\exp\{g(\theta)\cdot t(x)\}\cdot h(x) \]
