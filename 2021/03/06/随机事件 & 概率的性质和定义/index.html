随机事件&概率的性质和定义

Posted on 2021-03-06 Edited on 2021-03-08

随机事件

样本空间：随机现象的一切可能的基本结果的集合，记为\(\Omega = \{\omega\}\)，其中\(\omega\)表示基本结果，又称为样本点

集合按照元素个数可分为可数集和不可数集，其中：

  • 可数集：元素个数可数或可列
  • 不可数集：元素个数不可数或不可列

根据上述分类，样本空间又可以分为两类：离散样本空间（对应可数集）和连续样本空间（对应不可数集）

注：使用模型时需要注意对应的样本空间的类型

随机事件：随机现象的某些样本点组成的集合

随机事件可以分为以下几类：

  • 基本事件：由样本空间中的单个元素组成的子集
  • 必然事件：样本空间的最大子集
  • 不可能事件：样本空间的最小子集（空集）

以丢一个质量均匀的六面骰子为例，其中的样本空间\(\Omega\)={1,2,3,4,5,6}，为可数集，该样本空间为离散样本空间。

点数为奇数即为随机事件

点数为1为基本事件

点数为1~6内的某一个自然数为必然事件

点数为7为不可能事件

随机变量：用来表示随机现象结果的变量

事件的运算

事件之间的关系 集合关系
A发生必然导致B发生 \(A\subset B\)
A和B相等 \(A = B\)
A和B不可能同时发生 \(A\bigcap B = \varnothing\)
事件间的运算 集合运算
A与B至少发生一个 \(A\bigcup B\)
A与B同时发生 \(A\bigcap B\)
A不发生 \(\overline{A}\)（A的对立事件)
A发生但B不发生 \(A-B\)

事件间的运算性质：其中交、并运算等价

  • 交换律：\(A\bigcup B = B \bigcup A\)

  • 结合律：\(A \bigcup (B \bigcup C) = (A \bigcup B) \bigcup C\)
  • 分配律：\((A \bigcup B) \bigcap C = (A \bigcap C) \bigcup (B \bigcap C)\)

  • 对偶律：\(\overline{(A\bigcup B)} = \overline{A} \bigcap \overline{B}\)

事件域：设\(\Omega\)为样本空间，\(F\)为\(\Omega\)的某些子集所组成的集合类

​ 如果\(F\)满足以下条件：

​ (1). \(\Omega \in F\)

​ (2). 若\(A \in F\), 则 $ F$

​ (3). 若\(A_1, A_2, A_3, … \in F\), 则\(\displaystyle\bigcup_{i=1}^{\infty}A_i \in F\)

​ 则称\(F\)为一事件域，或\(\sigma\)域，或\(\sigma\)代数

概率

概率：设\(\Omega\)为样本空间，事件域\(F\)是由\(\Omega\)的一些子集所构成的。对任一事件\(A \in F\)，若定义在\(F\)上的一个实值函数\(P\)满足：

​ (1). 非负性：\(P(A) \geq 0\)

​ (2). 正则性：$P() = 1 $

​ (3). 可列可加性：对于互不相容的可列个事件序列\(A_1,A_2,…,A_n,…\)，有\(P(\displaystyle\bigcup_{i=1}^\infty A_i) = \displaystyle\sum_{i=1}^\infty P(A_i)\)

​ 则称\(P\)为概率，三元组\((\Omega, F, P)\)为概率空间

性质：

  • 零概率：\(P(\varnothing) = 0\)

  • 有限可加性：对于有限个互不相容的事件\(A_1,A_2,…,A_n\)，有\(P(\displaystyle\bigcup_{i=1}^n A_i) = \sum_{i=1}^n P(A_i)\)

  • 对立事件的概率：\(P(\overline A) = 1-P(A)\)

  • 单调性：若\(A \supset B\)，则\(P(A) \geq P(B)\)

    ​ 推论：\(P(A-B) = P(A) - P(AB)\)

  • 加法公式：\(P(A\bigcup B) = P(A) + P(B) - P(AB)\)

    ​ 若A和B不相容，则有\(P(AB) = P(\varnothing) = 0, P(A\bigcup B) = P(A) + P(B)\)

  • 连续性：

    • 为讨论连续性，给出如下关于极限事件的定义：

      若\(F_1 \subset F_2 \subset … \subset F_n \subset …\)，则有\(\bigcup_{i=1}^\infty F_n = lim_{n→\infty}F_n\)；

      其中\(\bigcup_{i=1}^\infty F_n\)称为\(\{F_n\}\)的极限事件。

      若\(E_1\supset E_2 \supset … \supset E_n \supset …\)，则有\(\bigcap_{i=1}^\infty E_n = lim_{n→\infty}E_n\);

      其中\(\bigcap_{i=1}^\infty E_n\)称为\(\{E_n\}\)的极限事件。

    • 下连续性：对事件域\(F\)上的一个概率\(P\)，若其对\(F\)中任一单调不减的事件序列\(\{F_n\}\)，有\(\displaystyle\lim_{n→\infty}P(F_n) = P(\displaystyle\lim_{n→\infty }F_n)\)，则称\(P\)是下连续的。

    • 上连续性：对事件域\(F\)上的一个概率\(P\)，若其对\(F\)中任一单调不增的事件序列\(\{E_n\}\)，有\(\displaystyle\lim_{n→\infty}P(E_n) = P(\displaystyle\lim_{n→\infty}E_n)\)，则称\(P\)是上连续的。

    • 性质：若\(P\)是定义在\(F\)上的一个概率，那么它既是上连续的，又是下连续的。

      证明：下连续性：

      ​ 设\(\{F_n\}\)是\(F\)中一个单调不减的事件序列，则\(F_1,F_2-F_1,…,F_n-F_{n-1},…\)，均为互不相容的集合。

      ​ 令\(F_0=\varnothing\)，则有: \[ F_n=(F_1-F_0)\bigcup(F_2-F_1)\bigcup …\bigcup(F_n-F_{n-1}) \]

      \[ P(F_n) = P(F_1-F_0) +P(F_2-F_1)+…+P(F_n-F_{n-1}) \]

      \[ P(F_n) = \sum_{i=1}^nP(F_i-F_{i-1}) \]

      ​ 根据极限性质，有： \[ P(\bigcup_{i=1}^\infty F_i) = \sum_{i=1}^\infty (F_i-F_{i-1}) = \lim_{n→\infty}\sum_{i=1}^nP(F_i-F_{i-1}) = \lim_{n→\infty}P(F_n) \] ​ 根据极限事件性质，有： \[ P(\bigcup_{i=1}^\infty F_i) = P(\lim_{n→\infty}F_n) \] ​ 上述两式可证： \[ \lim_{n→\infty}P(F_n)=P(\lim_{n→\infty}F_n) \] ​ 下连续性证毕。

      ​ 上连续性：

      ​ 设\(\{E_n\}\)是\(F\)中一个单调不增的序列，根据极限性质，有： \[ 1-\lim_{n→\infty}P(E_n) = \lim_{n→\infty}[1-P(E_n)] \] ​ 根据对立事件的概率，有： \[ \lim_{n→\infty}[1-P(E_n)] =\lim_{n→\infty}P(\overline {E_n}) \] ​ 此时\(\{\overline {E_n}\}\)为单调不减的序列，根据已证明的下连续性，有： \[ \lim_{n→\infty}P(\overline {E_n}) = P(\lim_{n→\infty}\overline{E_n}) \]

      ​ 根据极限事件性质，有： \[ P(\lim_{n→\infty}\overline{E_n}) = P(\bigcup_{n=1}^\infty \overline{E_n}) \] ​ 根据德摩根律（对偶律），有： \[ P(\bigcup_{n=1}^\infty \overline{E_n}) = P(\overline{\bigcap_{n=1}^\infty E_n}) \]

      ​ 又根据对立事件的概率，有： \[ P(\overline{\bigcap_{n=1}^\infty E_n}) = 1-P(\bigcap_{n=1}^\infty {E_n}) \]

      ​ 根据上述各式可得： \[ 1-\lim_{n→\infty}P(E_n) =1-P(\bigcap_{n=1}^\infty {E_n}) \]

      \[ \lim_{n→\infty}P(E_n) = P(\bigcap_{n=1}^\infty {E_n}) \]

      ​ 最后根据极限事件性质，有： \[ P(\bigcap_{n=1}^\infty {E_n}) = P(\lim_{n→\infty}E_n) \]

      ​ 上述两式可证： \[ \lim_{n→\infty}P(E_n)=P(\lim_{n→\infty}E_n) \] ​ 下连续性证毕。

      ​ 性质证毕。
